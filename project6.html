CENTRAL - Police Web App UI

For this project, I was tasked to create a UI for a police dispatch software. The software had to include functions for answering, cancelling and making calls. It also had to have functions for keeping track of and organising incoming calls, as well as being able to access and communicate with patrol cars with digital radio. The app has a process. First of all the user signs themselves in. The users name is then displayed in the top right along with a dropdown menu for modifying user settings. There is also a real-time clock. The user then either answers or makes a call then they can take notes of all the different aspects of the incident at hand, such as address, name, level of criticalness etc. Once inputed this information is parsed to the next tab where a map and a table with previous calls.
